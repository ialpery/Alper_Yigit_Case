# UI TEST OTOMASYONU CASE STUDY #
Önce HomePage, ardından QA sayfasına yönlenerek filtreleme yapan, iş ilanına tıklandığında doğru url'e yönlendiğini kontrol eden test senaryosu için yazılmış bir otomasyon projesidir.

Programlama dili olarak Java,
Otomasyon tool'u olarak Selenium
Test framework'ü olarak TestNG kullanılmıştır.
